== Description ==

Ace Header Footer is a WordPress plugin that lets you create custom headers and footers using Elementor. 
You can display them across the entire site or only on specific pages, giving full control over your layout.

Features:
* Create custom headers and footers with Elementor.
* Assign templates to the entire site or specific pages.
* Supports all major WordPress themes.
* Clean and optimized for performance.
* Works without coding.

== Installation ==

1. Upload the plugin folder `acewx-header-footer` to `/wp-content/plugins/`.
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Create custom headers or footers using the “Ace Header Footer” custom post type.
4. Select where to display them (entire site or specific pages).

== Frequently Asked Questions ==

= Can I use this plugin with any theme? =

Yes, it works with most themes including Storefront, Astra, Kadence, and OceanWP.

= Does it require Elementor Pro? =

No, it works with the free version of Elementor for basic layouts. Some advanced widgets may require Elementor Pro.
